# Gold Not Old 1.1
### Added "Adventure" World Type
This provides terrain in the style of the Adventure Update, with modern biomes, like the other World Types. Based on Moderner Beta's Release Hybrid preset.

### Renamed World Types
`Default` -> `Changed World`
`Classic` -> `Beta`

### Renamed Golden Days Base on download
Golden Days Base now downloads as `golden-days-base.zip`, as the file name changes prevent the resource pack order from applying correctly when updating.
As this is a file name change in itself, you'll have to manually adjust the resource pack order or update your configs through Config Manager.

### Updated all mods
All mods and resource packs have been updated.
