# Gold Not Old 1.0
First Public Release!

Gold Not Old provides a look and feel similar to old Minecraft, but with much of the new gameplay features retained!

This includes:
- PACP and Golden Days (with Polytone) to fill out every single gap with an older aesthetic!
- Default World Type based on 1.7-1.17.1 generation, with newer biomes
- Classic World Type based on b1.7.3 generation, also with newer biomes
- Old Villages with Gravel Paths and Workstation blocks
- Old Combat with shield blocking instead of sword blocking
- No Mob Latency included to make entities feel much more snappy, like in pre-1.3
- Sprint key is unbound by default, promoting the use of double-tap sprint
  - Since that's less convenient than just using Ctrl, it more or less promotes walking over running lol

Known issues:
- Horses have strange handling due to No Mob Latency
